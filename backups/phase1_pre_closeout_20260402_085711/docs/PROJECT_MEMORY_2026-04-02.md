## 13. Architecture Review On 2026-04-02

- A dedicated architecture review was added in `docs/APP_ARCHITECTURE_PROPOSAL_2026-04-02.md`.
- Core conclusion: the research pipeline folders are acceptable, but the application layer is missing, so `streamlit_app.py` has absorbed file IO, orchestration, snapshot assembly, and presentation concerns.
- Recommended direction: keep Streamlit for now, add a new `src/app` layer for repositories, services, view models, and pages, then progressively thin `streamlit_app.py`.
- Suggested first implementation slice:
  1. Extract config/report repositories.
  2. Extract watchlist service.
  3. Split the watchlist tab into an independent page module.
  4. Extract a shared holding snapshot service for `watch_plan` and `action_memo`.
